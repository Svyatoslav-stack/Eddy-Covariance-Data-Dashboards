Dashboards

Pulse Dashboard

Pulse dashboard displays daily raw HF and LF data from Agali II, Agali CC, Kauru, Soontaga, Palo Forest II, and Palo CC.
Dashboard updates every 30 minutes and resets at 01:00 (GMT+2).

Stations Dashboard

Stations dashboard displays weekly HF, LF, and EddyPro calculated data from Agali II, Agali CC, Kauru, Soontaga, Palo Forest II, and Palo CC.
Dashboard updates every day at 05:40 (GMT+2).
